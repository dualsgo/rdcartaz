# **App Name**: PromoPrint

## Core Features:

- Poster Content Input: Allows users to input product details such as description, original price, sale price, product code, and reference number.
- Real-time Poster Preview: Displays a live, dynamic preview of the poster as the user enters information, reflecting all design choices and content updates.
- Automated Discount Calculation: Automatically calculates and prominently displays the percentage discount based on the 'Price From' and 'Price For' inputs, only appearing when a discount is applicable.
- Direct Print Functionality: Enables users to print the generated poster directly from their browser, optimized for A4 landscape dimensions and minimal margins.
- AI-Powered Description Tool: Provides an AI tool to generate creative and compelling product descriptions or promotional taglines based on keywords provided by the user.

## Style Guidelines:

- Primary color: A bold and dynamic crimson red (#d51f2b) to signify urgency and promotional impact in a retail context. (HSL: 355, 75%, 48%)
- Background color: A subtly warm off-white (#fcf6f6), visually linked to the primary color but desaturated, ensuring a clean and light aesthetic that allows the poster content to pop. (HSL: 355, 18%, 98%)
- Accent color: A vibrant fuchsia (#ee83d3), chosen to provide high contrast and draw attention to call-to-actions and key informational snippets, analogous to the primary color yet distinct. (HSL: 325, 76%, 72%)
- Headlines and prominent text will use 'Poppins' (sans-serif) for its contemporary geometric style and strong visual presence, ideal for promotional messaging. Body text and numerical information will use 'Inter' (sans-serif) for optimal legibility and clarity, especially with varying text lengths and data.
- Utilize simple, modern, and clearly identifiable filled-style icons. These will provide intuitive navigation and indicate functionality, especially for print options and AI tools.
- The application will feature a responsive input form for optimal user experience across devices, leading to a fixed A4 landscape poster output with a clear visual hierarchy. Key information like prices and discounts will be emphasized through size and placement, mirroring traditional promotional posters.
- Incorporate subtle micro-animations such as gentle fades on text updates, a slight bounce on button presses, and smooth transitions when the discount percentage appears or changes, providing positive user feedback without distracting from the primary content.
